# コードレビューツール

新入社員のプログラミング学習を促進するためのAI活用コードレビューツールです。AIを活用して詳細なコードレビューを行い、新入社員が自身のコードの問題点を理解し、改善する能力を養成するとともに、管理者が新入社員の成長を追跡・評価できる機能を提供します。

## システム概要

このアプリケーションは以下の特徴を持っています：

- AIを活用した詳細なコードレビュー
- 新入社員向けのコード改善フィードバック
- レビュー内容についてのAIチャットサポート
- 管理者向けの評価・進捗管理機能
- Backlog連携による実プロジェクトへの反映
- 詳細な分析ダッシュボードによる成長追跡

## 主な機能

### 新入社員向け機能

1. **コードレビュー依頼**
   - コードの提出と自動AIレビュー
   - 詳細なフィードバックと改善提案
   - 優先度付きの問題点指摘

2. **フィードバック対応**
   - 問題点ごとの対応状況管理
   - AIアシスタントによる質問対応
   - 修正コードの再提出と進捗確認

3. **プルリクエスト連携**
   - Backlogへのコード変更適用
   - レビュー結果をプルリクエストにコメント

4. **パフォーマンス確認**
   - スキルレベル(A～E)の可視化
   - 過去レビュー履歴と成長推移の確認
   - スキル要素別の評価グラフ

### 管理者向け機能

1. **社員管理**
   - 年度ごとの新入社員一覧
   - 部署・入社年度によるフィルタリング
   - 社員詳細情報の管理

2. **評価分析**
   - スキルレベル分布のグラフ表示
   - フィードバックタイプ別統計
   - 成長推移の可視化
   - 新入社員ランキング

3. **リポジトリ管理**
   - Backlogリポジトリの登録と管理
   - 自動コードレビュー対象の設定
   - Webhook設定とテスト

## 開発環境のセットアップ

### 前提条件
- Docker と Docker Compose がインストールされていること
- Node.js 18以上がインストールされていること
- OpenAI APIキーの取得
- Backlog 公開鍵設定(https://support-ja.backlog.com/hc/ja/articles/360036145053-Git%E3%81%AE%E6%A6%82%E8%A6%81#index4)

### セットアップ手順

1. リポジトリをクローン
   ```bash
   git clone xon@xon.git.backlog.jp:/AD_TECHNOLOGY/pluto.git
   cd pluto
   ```

2. 環境変数ファイルの作成
   ```bash
   cp backend/.env.example backend/.env
   cp frontend/.env.example frontend/.env
   ```

3. 環境変数の設定
   ```
   # backend/.env
   OPENAI_API_KEY=your_openai_api_key
   BACKLOG_API_KEY=your_backlog_api_key
   BACKLOG_SPACE=your_backlog_space
   ```

4. Docker Composeでアプリケーションを起動
   ```bash
   // backendサーバーの起動
   cd backend 
   npm run dev

   // frontendサーバーの起動(別ターミナル起動)
   cd frontend
   npm run dev
   ```

5. アプリケーションにアクセス
   - フロントエンド: http://localhost:3000
   - バックエンドAPI: http://localhost:3001
   - Chroma Vector DB: http://localhost:8000
   - PHPMyAdmin: http://localhost:8080

## 利用ガイド

### 新入社員向け

1. アカウント登録・ログイン
2. ダッシュボードからレビュー依頼を作成
3. コードを提出してAIレビューを受ける
4. フィードバックに対応して修正コードを提出
5. 必要に応じてAIチャットで質問

### 管理者向け

1. 社員管理画面で新入社員を管理
2. 分析ダッシュボードで全体進捗を確認
3. リポジトリ管理でBacklog連携を設定
4. 個別社員の詳細分析を確認
5. 設定画面でシステム動作をカスタマイズ

## 将来展望

本システムは以下の機能強化を予定しています：

### 外部システム連携の強化
- **GitHub連携の追加**: BacklogだけでなくGitHubとの連携機能を実装し、より広い開発環境をサポート

### 学習促進機能の拡充
- **リファレンスナレッジベース**: 問題点に関連したドキュメントやベストプラクティスへのリンク提供
- **ヒント段階提供**: 全ての答えを一度に表示せず、段階的なヒントでスキルアップを支援
- **演習・課題単位の管理**: 特定の学習目標に沿った課題セットの管理機能
- **スキルツリーの導入**: 習得すべきスキルを可視化し、進捗を体系的に管理

### 学習モチベーション向上
- **ゲーミフィケーション**: バッジやレベルアップシステムの導入
- **採点システム**: 各レビューに対する点数評価とランキングの実装
- ~~**使用制限の設定**: レビュー回数やチャット利用回数の制限による計画的利用促進~~

### データ分析・エクスポート
- **詳細レポート機能**: 月次・年次データのExcel/PDF形式エクスポート
- **カスタムダッシュボード**: 分析観点のカスタマイズと保存
- **成長予測モデル**: AIによる成長予測とアドバイス生成
- **マネジメントレポート**: 経営層向けの要約レポート生成

### AIレビュー品質向上
- **プロンプト最適化**: レビュー精度向上のためのプロンプト継続改善
- **独自モデルのファインチューニング**: コードレビュー特化型のモデル開発
- **言語別・フレームワーク別のレビュー**: 特定技術に最適化されたレビュー提供
- **過去レビュー学習**: 過去のレビュー結果を学習し、個人に合わせたフィードバック生成

これらの機能強化により、新入社員の成長をさらに効果的に促進し、企業全体のエンジニアリング品質向上に貢献します。
